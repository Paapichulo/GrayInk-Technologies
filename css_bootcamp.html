Course Description

CSS acronym for Cascading Style Sheet, this defines the stylish outlook of a website. It is responsible for making websites presentable with appealing User Interfaces(UI) and compatibility with all screen devices (Desktop, Laptops, Tablets, Phablets, Phones) and browsers. This consists of colors, fonts, layouts etc.

Course Content

  • CSS Colors
  • CSS Background
  • CSS Selectors
  • CSS Variables
  • CSS Units
  • CSS Math Functions
  • CSS Layouts(grid, flexbox, floats, positions)
  • CSS Box model
  • CSS Events
  • CSS Fonts
  • CSS Psuedo elements and Classes
  • CSS Perspectives (2D, 3D, transforms)
  • CSS Animations
  • CSS Responsiveness (media query, viewport) etc.
  • etc...
  • CSS Quiz
