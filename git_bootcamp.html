Course Description

GIT is a distributed version control system for managing files, tracking changes and storing different source code in software development. It helps in coordinating work among programmers collaboratively. Its goals include speed, data integrity, and support for distributed, non-linear workflows through its thousand of parallel branches.


Features Include:

  • Free and open source
  • Tracks work history
  • Creates file backups
  • Supports non-linear development & collaboration
  • Scalable

  • Git Conflicts

Course Aim

Students should be able to:

  • Set up git environment
  • Understand the basic commands and workflow
  • Resolve git conflicts
  • Store, manage and track file changings
  • Work collaboratively with other developers.
